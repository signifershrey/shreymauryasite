Web Hosting on Netlify

What do you mean by Web Hosting?

Web hosting is like a place on internet where all the files of your website are living and they can be accessed from anywhere all over the world.
It is like the home of your website to store the flies you related to your website(code,image,etc).

Why you should use Netlify for Web Hosting?

Netlify offers Free Hosting and Serverless Backend services for Web Applications and static websites.
For a Begineer to start with Hosting ,Netlify is very Easy to use also it has features like building fast modern websites,Continuous deployment,Branch Deploys, and much more.
Some of the Best features are -

  • Deploy in seconds

    Deploying projects on Netlify is easy and fast.It provides the options to deploy from a Git repository or just by simply dragging your project folder and dropping it on the site area.

  • Continuous Deployement

    Whenever we deploy a project from Git repository,It automatically acitvates CD for our project so whenever we make a new commmit and push the repository ,Netlify automatically updates the project.

  • Deploy Previews

    Netlify also provides a option to preview every deploy you want to make, without actually having to deploy them in our existing project site.

  • Custom Subdomains

    By default,Netlify gives you a free custom subdomains whenever you create a new project,but it also provides a option to use a custom domain name.You can go to site change settings and change your site name.

  • Free HTTPS (SSL)

    Netlify offers free HTTPS on all sites, including automatic certificate creation and renewal. Click For more details
